# Audio Files / Lydfiler

Læg dine julemusik-filer i denne mappe.

## Understøttede formater:
- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)
- M4A (.m4a)

## Sådan virker det:
1. Læg lydfilerne i denne mappe
2. Filerne vil automatisk blive afspillet i alfabetisk rækkefølge
3. Når en sang slutter, starter den næste automatisk
4. Når alle sange er afspillet, starter playlisten forfra
5. Musikken er skjult og afspilles automatisk som baggrundsmusik ved 30% volume

## Eksempel:
```
public/audio/
├── 01-jingle-bells.mp3
├── 02-silent-night.mp3
├── 03-white-christmas.mp3
└── README.md
```

Musikken starter automatisk når siden loader.
